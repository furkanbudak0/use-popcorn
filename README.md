# usePopcorn

React ile geliştirilmiş, film arama ve izleme listesi uygulaması.

## Özellikler

- OMDb API ile film arama
- İzlenen filmleri listeleme ve puanlama
- Modern ve responsive arayüz

## Kurulum

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
2. Uygulamayı başlatın:
   ```bash
   npm start
   ```
